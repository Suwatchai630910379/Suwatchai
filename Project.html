12/24V militaly

          The 12/24 V Military Trailer Module It is a module used for controlling the operation of light signals. (For example, Control the operation of the turn Left Right signal,revert lamp,tail lamp within the trailer module. This upcoming module will utilize the CAN BUS Protocol for communication with the controller, specifically the STM32F105VCT6. It will have the capability to both receive and transmit CAN BUS data within the vehicle and will leverage STM32 software to facilitate the analysis of various data points.

Fish-Tank Supervision By Line Bot

          The Fish-Tank Supervision Project employed Python and Arduino IDE to create a sophisticated control system accessible through a LINE BOT interface. This innovative project offers three key functionalities: the ability to toggle the aquarium light, facilitate feeding, and monitor water temperature. By seamlessly integrating these features, it not only provides a user-friendly interface for fish tank management but also ensures real-time monitoring and control, showcasing the versatility and practicality of combining Python, Arduino, and LINE BOT technology for efficient and convenient aquarium maintenance.
